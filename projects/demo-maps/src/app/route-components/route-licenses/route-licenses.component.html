Licenses setup

This component expects a assets/licenses.json file to be present in the project bundle.

That file was created with the following command:

cd ./frontend-libraries npm install -g license-checker npx license-checker --json --out ./dist/demo-maps/assets You might want to add this command to the build-process by adding a "postbuild" task to your package.json.
