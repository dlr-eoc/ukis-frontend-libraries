This example application shows how to work with the ukis-frontend-libraries

Each card brings you to a route component rendered inside a `router-outlet`. The components have a `content-container` class binding to make the layout structure of Clarity Application Layout correct. Each map component uses providers for `LayersService`, `MapStateService`... so they are encapsulated and not shared between all the routes.
