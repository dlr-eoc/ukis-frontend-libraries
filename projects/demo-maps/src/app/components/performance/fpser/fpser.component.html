Frames per second

Estimates the number of frames that are rendered per second.

This is done by periodically measuring the time a call to setTimeout(() => ..., 0) takes.

That call is made outside the angular-zone, so as not to trigger change-detection.

Please not that in Chrome you can also monitor FPS from the developer-tools: Hit ctrl-shift-p and enter fps to find the FPS-meter.
